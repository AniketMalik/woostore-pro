Install Plugins

In this guide you will install some required and optional wordpress plugins for your WooStore Pro mobile application.

note

Installing a wrong plugin might break the application processes. So please pay attention to the plugin and author names.

Required Plugins#

info

Below mentioned plugins are Required and must be installed before trying to run the application.

Below are mentioned the plugin names and author names which we will use to set up the application. Please install the correct plugin by checking the plugin name and author name.

Some of the plugins which you will install require additional configuration. We will configure all the plugins in the next section. In this section we will only install and activate them.

How to install plugins from the marketplace ?#

  1. Go to Plugins Installer.
  2. Click on Add New on the top left corner.
  3. Type the name of the plugin in the search filed (on the right) and wait for the results.
  4. Match the plugin and author name. For your reference the images for plugins are given below.
  5. Click on Install and then Activate.

8. WooStore Pro Api#

By Aniket Malik

This is the plugin which you got in your download package with the full application source code. This plugin package can be found at Download Package --> wordpress-plugins --> woostore-pro-api.zip

How to install this plugin ?#

  1. Go to Plugins Installer.
  2. Click on Add New on the top left corner.
  3. Click on Upload Plugin on the top left corner.
  4. Click on the Choose file button.
  5. Locate the Download Package --> wordpress-plugins --> woostore-pro-api.zip file.
  6. Click on Install Now and if prompted Activate the plugin.

This will install the WooStore Pro Api Plugin in your wordpress installation for the application to connect to your website.

info

If you are using WooStore Pro Api wordpress plugin version v2.0.0 or above, you can proceed to next step. For all previous version you need to add 2 more plugins.

If you are using application version v1.9.4 you might be using WooStore Pro Api - v2.0.0 and no additional steps are required. You can proceed to next steps.

You can check the WooStore Pro Api wordpress plugin version from the download package / wordpress-plugins / woostore_pro_api.zip file. You will need to unzip / exract the plugin file and look at the change_logs.md. The top most entry about the version is the version you have.

  • Plugin version v2.0.0 and above
  • All Previous Plugin Versions
