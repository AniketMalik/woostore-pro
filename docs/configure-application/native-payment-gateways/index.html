Native Payment Gateways

Please use the following guide to use the supported native payment gateways in the application.

Requirements#

  1. WooStore Pro mobile application version v3.5.0 or above.

  2. WooStore Pro Api wordpress plugin version v4.5.0 or above.

  3. App Builder wordpress plugin version v1.5.0 or above.

Supported Plugins#

  1. Paypal
  2. Stripe
  3. Flutterwave
  4. Paystack
  5. Paytm
  6. Razorpay

How to use it ?#

  1. Open the App Builder
  2. Select your active template and click on customize.
  3. Go to Global Configurations --> Native Payment Gateways.

  1. Enable the payment gateway that you want to use in the application and add the respective credentials by clicking the down arrow.
