Product Categories

You can modify some of the settings related to product categories to change how they appear in the application.

tip

Checkout the new layout options for categories screen: Categories Screen Layouts

In this section you will be able to modify how the Product Categories are displayed in the application.

info

All the flags that you see here are defined in the Config.dart file. You can locate this file in woostore_pro / lib / constants / config.dart

Categories List In Home Screen#

As of version v1.7.0 new layout settings has been added to modify the categories list in home screen.

New#

  • categoriesHomeScreenLayout

    This flag allows you to change the layout of the categories list in the home screen. The available options are grid and list-horizontal

  • categoriesHomeScreenItemBorderRadius

    This setting can be used to change the shape of the items. A value >= 50 will make the items circular

  • categoriesHomeScreenGridLayoutCrossAxisCount

    This flag allows you to change the number of grid items you want to display in the single row. This setting will only work if categoriesHomeScreenLayout is set to grid.

    Checkout the gif below to get a better idea on how you can change the layout of the categories list in the home screen.

Previous#

  • showCategoriesInHome

    This flag is used to set the visibility of the product categories in the Home Screen. If this is set to false, the home screen will not show categories.

Other Settings#

Below are setting flags for other screens and places where the categories are used.

  • showCategoriesInSearchFilterModal

    This flag is used to hide/show the categories filters in the search filter options

  • showCategoriesInAllProductsFilterModal

    This flag is used to hide/show the categories filters in the all products screen filter modal. This is the screen the user is navigated to when he clicks on any of the home sections.
